    Java 大数值操作（BigInter、BigDecimal）

    Posted on 2021-05-02

    Java大数值

    可以使用BigInteger操作大整数
    可以使用BigDecimal指定小数的保留位数

    大数操作

    BigInter

    如果在操作的时候一个整型数据已经超过了整数的最大类型长度long的话，则此数据就无法装入，所以，此时要使用BigInteger类进行操作。

    No. 方法 描述
    1 BigInter(Strring val) 将一个字符串转化为BigInter类型
    2 BigIner add(BigInter val) 加法
    3 BigIner substract(BigInter val) 减法
    4 BigIner multiply(BigInter val) 乘法
    5 BigIner divide(BigInter val) 除法
    6 BigIner max(BigInter val) 返回两个大数字中的最大值
    7 BigIner min(BigInter val) 返回两个大数字中的最小值
